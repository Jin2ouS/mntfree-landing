# MnTfree Landing

**MnTfree - 시간을 사는 자동화**  
반복을 시스템으로 바꾸면, 자유는 시간 위에 쌓입니다.

Glassmorphism 스타일의 MnTfree 랜딩 페이지입니다. React + TypeScript + Vite로 구성되어 있으며, Stock / Crypto / Futures / Property 서비스로 연결됩니다.

## 서비스 링크

| 카테고리 | 주소 |
|----------|------|
| Stock | [stock.mntfree.com](https://stock.mntfree.com) |
| Crypto | [crypto.mntfree.com](https://crypto.mntfree.com) |
| Futures | [futures.mntfree.com](https://futures.mntfree.com) |
| Property | [property.mntfree.com](https://property.mntfree.com) |

## 로컬 실행

**필수:** Node.js

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (기본: http://localhost:3000)
npm run dev
```

## 빌드

```bash
npm run build
```

산출물은 `dist/` 폴더에 생성됩니다. 미리보기는 `npm run preview`로 확인할 수 있습니다.

## 기술 스택

- React 19, TypeScript
- Vite 6
- Tailwind CSS (CDN)
